# Sudoku Solver

A sudoku solver in Haskell. 

## How to install

    $ git clone https://github.com/thomasbrus/sudoku-solver.git
    $ cd sudoku-solver
    $ cabal install --prefix=$HOME --user

Run with:

    $ ~/bin/sudoku-solver
